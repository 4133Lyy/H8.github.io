jQuery 是一个快速、简洁的 JavaScript 框架，诞生于 2005 年，由 John Resign 开发。 它封装 JavaScript 常用的功能代码，提供一种简便的 JavaScript 设计模式，优化 HTML 文档操作、事件处理、动画设计和 Ajax 交互。 jQuery 是目前最受欢迎的 JavaScript 框架。它使用 CSS 选择器来访问和使用 jQuery：。
jQuery 库包含的功能包括：HTML 元素选取、HTML 元素操作、CSS 操作、HTML 事件函数、JavaScript 特效和动画、HTML DOM 遍历和修改、AJAX 异步请求方式等。
